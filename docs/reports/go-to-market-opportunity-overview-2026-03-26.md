# Example Query Output

## Prompt

Which underappreciated Steam games are showing early signs of a commercial push, and which ones still look signable before the breakout is obvious?

## Assumptions Used

This output is based on four simple assumptions:

- Reviews should be clearly positive.
- Reviews should still be coming in, or the recent review trend should be improving.
- The team should have made visible public changes in the March 14-25, 2026 window, such as game updates, store-page edits, pricing changes, or announcements.
- The game should not already be wrapped in a large publisher setup.
