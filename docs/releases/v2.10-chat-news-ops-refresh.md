# v2.10 - Chat News, CCU Quality, and Ops Refresh

**Release Date:** March 30, 2026

Version 2.10 tightens the current release surface around recent-news chat, operational visibility, and change-intelligence plumbing:

- recent Steam news now has dedicated latest-item detail, bounded digest, and topic-search tools
- admin operations now surface catalog control and CCU quality, with cached stats for the main dashboard
- PICS gained prioritized first-pass sync for newly discovered unsynced apps
- news search projections and app-capture work state keep the current change-intelligence runtime responsive

---

## Highlights

### Recent News in Chat

- `get_recent_news_detail` now answers "what actually changed in the latest Steam news" for a title.
- `get_recent_news_digest` supports one title or a small explicit title set with bounded item budgets.
- `search_recent_news_topics` searches stored news text for concrete topics like developer diaries, patch notes, demos, playtests, and roadmaps.
- redundant recent-news calls are blocked within a turn when the earlier result is already sufficient to answer.

### Change Intelligence and Admin Ops

- `app_capture_work_state` replaces the older work tracking path for storefront/news/hero-asset capture.
- `news-hot-refresh.yml` and bounded news catch-up flows now reuse the same queue worker instead of broad rescans.
- the admin dashboard now exposes catalog control stats, CCU quality stats, and richer chat-log quality metadata.
- CCU quality stats are cached so the dashboard can show confidence, source mix, and suspect-zero breakdowns without expensive recomputation.

### PICS and Ingestion

- `MODE=first_pass` gives the PICS service a prioritized bounded backfill path for newly discovered unsynced apps.
- PICS history capture still writes normalized snapshots and diffs before latest-state upserts.
- Storefront remains authoritative for parsed `release_date` and `is_free`; PICS continues to act as enrichment and fallback data.

---

## Notable Runtime Surfaces

- `/chat` now has recent-news detail, digest, topic-search, before/after comparison, and deterministic change-pattern tools available through the chat runtime.
- `/admin` now includes catalog control and CCU quality sections alongside the existing sync, PICS, jobs, and chat-log views.
- `/changes` remains the unified Steam Activity surface for grouped bursts, announcements, and health states.
